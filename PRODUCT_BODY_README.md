# 製品本体

利用者向けの製品本体は次の2点です。

1. `spreadsheet/SIMS-Blog-Manager.xlsx`
2. `apps-script/Code.gs`

Apps Scriptの権限が必要な場合は `apps-script/appsscript.json` も反映してください。

## 今回の確認ポイント

- データ一覧のH1タイトル/titleタグ/meta descriptionに、記事IDのような数字が表示されないこと
- クリック数・表示回数が3桁区切りで表示されること
- CTRが小数第1位の％表示になること
- 平均順位が小数第1位で表示されること
