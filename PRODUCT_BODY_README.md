# 製品本体

Product 5.0 Official の製品本体は以下です。

- `spreadsheet/SIMS-Blog-Manager.xlsx`
- `apps-script/Code.gs`

今回の版では、STEP B「改善候補を分析」を現行機能だけに限定しています。

実行する処理:

1. Search Console取得済みデータを読む
2. 改善候補を分析する
3. 今日の改善を更新する
4. 改善中を更新する
5. データ一覧を状態順で更新する
6. Homeを更新する
7. 処理ログを記録する

現段階で未実装として外した機能:

- 記事ネタ候補
- カニバリ診断
- 上位ページ診断
- 効果測定
- 測定履歴

### 今回の確認ポイント

1. `apps-script/Code.gs` を入れ替えてください。
2. STEP B「改善候補を分析」を実行してください。
3. 実行中に `記事ネタ候補` シートが一瞬でも表示されないことを確認してください。
4. `改善ブリーフ` の記事タイトルがURLではなく、記事タイトル風の表示になっていることを確認してください。
