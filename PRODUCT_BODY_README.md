# Product Body

製品本体は次の2つです。

- `spreadsheet/SIMS-Blog-Manager.xlsx`
- `apps-script/Code.gs`

## 今回の確認ポイント

1. Apps Scriptの `Code.gs` を入れ替えます。
2. STEP Aを実行します。
3. `#` 付きURLが記事本体URLへ統合されることを確認します。
4. データ一覧で「記事タイトル」「SEOタイトル（titleタグ）」「メタディスクリプション」が表示されることを確認します。

記事タイトル取得は1回あたりの取得件数上限に従い、未取得URLを少しずつ補完します。
