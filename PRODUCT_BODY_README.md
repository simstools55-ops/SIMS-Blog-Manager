# 製品本体

- Apps Script: `apps-script/Code.gs`
- Manifest: `apps-script/appsscript.json`（今回は更新不要）

現在のスプレッドシートを継続使用できます。
`管理 → シートを作成・修復` を1回実行すると、既存記事DBを記事ランク＋作業状態へ移行します。


## 今回の運用変更

- Homeは記事ランクと作業状態を記事DBから集計します。
- 1日の最初の起動時に日次更新の実行確認を表示します。
- 日次更新に出なかった記事は削除せず、内部で未取得日数を管理します。
- 新規記事が追加された場合は記事情報補完を促します。
