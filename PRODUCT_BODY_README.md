# 製品本体

利用者が更新する製品本体は次のファイルです。

- `apps-script/Code.gs`

## 更新手順

1. Apps Scriptの既存 `Code.gs` を全置換します。
2. 保存後、スプレッドシートを再読み込みします。
3. `管理 → シートを作成・修復` を1回実行します。
4. 「データ一覧」と旧SearchConsole_Dataが削除され、記事DB中心の構成になったことを確認します。

## 継続利用するシート

- Home
- 記事DB
- 今日の改善
- 改善ブリーフ
- 改善中
- 設定
- 処理ログ

## 保留中

- 効果測定
- 測定履歴

現行の今日の改善を壊さないため、内部のブログ診断データは非表示で一時保持します。今日の改善を記事DB直結方式へ変更した後に削除します。
