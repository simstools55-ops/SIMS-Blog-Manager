# Product 5.0 ArticleDB Compact & Home Insight

記事DBを日常運用向けにコンパクト化し、Homeに記事ランクの構成・前回差分・総合コメントを表示する更新です。

## 主な変更

- 日次更新の説明を現行仕様へ更新
- 記事DBの並び順を「エース、安定、成長、育成、低迷」に変更
- 記事DB上では記事ランク、作業状態、URL、メインクエリ、指標、記事タイトル、記事詳細だけを表示
- 詳細セルを選択すると、SEOタイトル、メタディスクリプション、管理情報をポップアップ表示
- Homeに記事ランク件数の前回差分とブログの現在地コメントを表示

## 更新方法

`apps-script/Code.gs` を全置換し、スプレッドシートを再読み込み後、管理メニューの「シートを作成・修復」を1回実行してください。
