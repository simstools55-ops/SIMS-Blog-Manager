# 製品本体の場所

このZIPで利用者が使う製品本体は次の2つです。

1. `spreadsheet/SIMS-Blog-Manager.xlsx`
2. `apps-script/Code.gs`

GoogleスプレッドシートへExcelをアップロードし、Apps Scriptに `Code.gs` の中身を貼り付けて使います。

## Product 5.0の表示方針

利用者が使う画面だけを残します。

- Home
- データ一覧
- 今日の改善
- 改善中
- ブログ診断
- 処理ログ
- 改善ブリーフ

上位ページ診断・カニバリ診断など、現段階で使わない機能は削除対象です。
