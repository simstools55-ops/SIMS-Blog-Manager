# Product 5.0 Official 製品本体

このZIPの製品本体は次の2つです。

## 1. スプレッドシート

`spreadsheet/SIMS-Blog-Manager.xlsx`

Googleドライブへアップロードし、Googleスプレッドシートとして開いて使用します。

利用者に見えるタブは次の5つだけです。

1. Home
2. 今日の改善
3. 改善中
4. ブログ診断
5. 処理ログ

以下の管理用シートは通常表示しません。

- 設定
- SearchConsole_Data
- Improvement_Queue
- Improvement_Log

## 2. Apps Script

`apps-script/コード.gs`

Apps Scriptの `コード.gs` に貼り付けて使用します。

Product 5.0 Officialでは、利用者向けスクリプトはこの1ファイルに集約します。
