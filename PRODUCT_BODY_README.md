# Product 5.0 Official 製品本体の場所

このZIPが、SIMS-Blog-Manager Product 5.0 Official の製品本体です。

## 利用者が使うもの

1. スプレッドシート本体

`spreadsheet/SIMS-Blog-Manager-template-Product5.0-Official-Lean.xlsx`

Google Driveへアップロードし、Googleスプレッドシートとして開いて使います。

2. Apps Script

`apps-script/コード.gs`

Googleスプレッドシートの「拡張機能 → Apps Script」を開き、既存のコードへ貼り付けます。

3. マニュアル

`docs/`

GitHub Pagesで公開する利用者向けマニュアルです。

## Product 5.0 の配布方針

Product 5.0では、マニュアルサイトから無理にテンプレートを直接配布しません。
製品本体はZIPで管理し、利用者または管理者はこのZIPから必要ファイルを取り出します。

## 最小利用手順

1. ZIPを展開する
2. `spreadsheet/` 内のExcelテンプレートをGoogle Driveへアップロードする
3. Googleスプレッドシートとして開く
4. `apps-script/コード.gs` をApps Scriptへ貼り付ける
5. 初回セットアップを実行する
