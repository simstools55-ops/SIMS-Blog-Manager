# 製品本体

利用者が更新する製品本体は次のファイルです。

- `apps-script/Code.gs`

## 更新手順

1. Apps Scriptの既存 `Code.gs` を全置換します。
2. 保存後、スプレッドシートを再読み込みします。
3. `管理 → シートを作成・修復` を1回実行します。
4. 「設定」シートで各数値を確認・変更します。

## 設定項目

- 記事情報補完件数：30～100（初期値50）
- 今日の改善初期表示件数：1～6（初期値2）
- 今日の改善最大表示件数：2～20（初期値6）
- 改善候補抽出件数：10～200（初期値50）
- Search Console取得期間：7～365日（初期値90日）
