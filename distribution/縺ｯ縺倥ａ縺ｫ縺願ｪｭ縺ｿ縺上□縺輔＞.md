# SIMS-Blog-Manager Product 5.2.0 Official

このフォルダには利用開始に必要なファイルだけが入っています。

- `コード.gs`
- `appsscript.json`
- `SIMS-Blog-Manager-template-Product5.2-Official.xlsx`

## 初回導入

1. テンプレートをGoogleドライブへアップロードし、Googleスプレッドシートとして開きます。
2. **拡張機能 → Apps Script**を開き、`コード.gs`を全置換します。
3. プロジェクトの設定でマニフェストを表示し、`appsscript.json`を反映します。
4. スプレッドシートを再読み込みします。
5. **SIMS-Blog-Manager → シートの作成・修復**を1回実行します。
6. **SIMS-Blog-Manager → 初回セットアップ**を実行します。

## 毎日の流れ

1. Homeでブログの様子を確認
2. **記事改善スタート**で改善記事を選択
3. 改善後に**結果登録**
4. **推移確認**で7日・14日・21日・28日の変化を確認
5. 必要に応じて**記事一覧**や**改善履歴**を確認

既存スプレッドシートは継続利用できます。更新時も`コード.gs`を入れ替え、シートの作成・修復を1回実行してください。
