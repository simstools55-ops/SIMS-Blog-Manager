# SIMS-Blog-Manager Product 5.2.1 Official

SIMS-Coreへ渡す改善依頼文を強化しました。

## 主な変更

- Search Console上位20クエリを、クリック・表示回数・CTR・平均順位付きで追加
- 改善優先順位、記事ランク、変更方針を依頼文へ追加
- 内部リンク候補を、タイトル・URL・推奨アンカー・関連クエリ・関連度で表示
- 内部リンク候補を採用・保留・不採用に分類するようSIMS-Coreへ依頼
- テキストリンクはHTML埋め込み済みのコピペ可能な完成形で出力するよう明記
- ブログカードは挿入位置、URL、記事タイトル、採用理由を出力
- カニバリ候補、記事統合判定、記事役割分担は今回未実装

## 更新方法

1. `コード.gs`を全置換して保存
2. スプレッドシートを再読み込み
3. バージョン情報が`5.2.1`であることを確認

シート構造は変更していないため、通常は「シートの作成・修復」は不要です。

- クエリ取得中も元のシートを維持し、改善ナビ内で取得内容を確認可能
