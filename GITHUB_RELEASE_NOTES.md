# Product 5.1 Official

## 主な変更

- Homeを記事ランク、今日のメッセージ、改善状況、今週の取り組み、今週のアドバイスの順に再構成
- 改善状況と今週の取り組みを左右2列で表示
- 「効果測定中」を「改善推移確認中」へ統一
- 「改善効果」シートを「改善の推移」へ変更
- 未取得記事の説明をHomeへ追加
- 改善履歴を開く際に、一覧再構築・書式・チェックボックスを再適用
- 既存スプレッドシートを継続使用可能

# SIMS-Blog-Manager Product 5.1 Official

正式バージョン `5.0.1` を公開します。

## 主な内容

- Search Consoleデータから今日改善する記事を選定
- 改善計画・実施内容・改善の推移を一つの詳細画面で確認
- 7日・14日・21日・28日の4週間効果測定
- Home、セットアップ、記事管理の初心者向けUI
- RC3 Performanceで確認した高速なメニュー応答

## 新規利用

添付の配布ZIPを展開し、「はじめにお読みください.md」の手順に従ってください。

## RC3 Performanceからの更新

既存スプレッドシートをそのまま利用できます。`コード.gs` を正式版へ全置換し、スプレッドシートを再読み込みしてください。

- 新しいスプレッドシート：不要
- 「シートの作成・修復」：不要
- バージョン表示：`5.0.1`
