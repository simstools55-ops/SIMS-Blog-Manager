# Changelog

## Product 5.0 RC1

### Added
- Search Consoleクエリ分類機能を追加。
- 最大50件のクエリを、サブクエリ・FAQ候補・別記事候補・除外クエリへ分類。
- カニバリ診断シートを追加。
- カニバリ発生時の対応アドバイスを追加。
- 改善ブリーフに「別記事候補」「除外クエリ」「カニバリ診断」を追加。

### Improved
- 改善ブリーフをClaude / ChatGPTへ貼り付けやすい形式へ改善。
- 測定履歴の記事タイトルがURLになる問題を改善。
- 順位、CTR、クリック数、表示回数の表示形式を統一。

### Notes
- Product 5.0 RC1ではSERP分析やライバルチェックは行いません。
- SERP分析・競合分析・記事執筆はSIMS-Core側の役割とし、Blog ManagerはSearch Consoleデータの整理・分類・管理に集中します。
