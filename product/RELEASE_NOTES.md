## Product 5.0 Official — 2026年7月13日

正式バージョン：`5.0.0`

### 主な機能

- Google Search Consoleデータの日次取得
- 優先度に基づく「今日の改善」表示
- 記事管理と改善ナビ
- 改善計画・実施内容・改善の推移を統合した詳細表示
- 7日・14日・21日・28日の4週間効果測定
- Home画面の処理状況表示
- 初心者向けセットアップウィザード

### 正式版での整理

- RC3 Performance実機テスト合格版を正式版として確定
- バージョン表記を `5.0.0` に統一
- Apps Script配布を `コード.gs` 1ファイルに統一
- 過去版テンプレートと開発用RC資料を利用者向け配布物から除外
- GitHub Pagesマニュアルを正式版の画面と運用に更新

### RC3 Performanceからの更新

既存スプレッドシートを継続利用できます。正式版の `コード.gs` へ全置換後、スプレッドシートを再読み込みしてください。「シートの作成・修復」は不要です。

## 5.3.2 Home rank and detail action restoration

- Homeのランク表示を左側「エース・安定・成長」、右側「育成・迷走・未取得」の3行2列へ変更。
- 未取得は一時欠落ではなく、3回連続または14日以上の要確認記事のみ集計。
- 改善効果シートを「改善の推移」へ改称し、旧名称から自動移行。
- 今日の改善、改善の推移、記事管理、改善履歴の選択行詳細操作をメニューへ復元。
- Search Console照合件数が既存記事の20%未満の場合、全記事を未取得扱いにしない安全装置を追加。
