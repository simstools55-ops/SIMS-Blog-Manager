# Product 5.0 Today Improvement V1

記事DB直結の「今日の改善」と「改善ナビ」を実装しました。

## 主な機能

- 即効性が高い記事：上位3件
- CTR改善効果が高い記事：上位3件
- 初期表示：各分類1件、合計2件
- 追加表示：2件ずつ、最大6件
- 改善理由・期待効果・予想時間を一覧表示
- 選択記事の改善ナビをポップアップ表示

## 操作

上部メニュー「今日の改善操作」から作成、追加表示、改善ナビ表示を行います。

# Product 5.0 ArticleDB Daily Diff Fix

日次更新を記事DBの差分更新方式へ変更しました。

## 日次更新で変更する項目

- 記事ステータス
- クリック数
- 表示回数
- CTR
- 掲載順位
- 最終取得日時
- 元URL件数

## 保護する項目

- ArticleID
- 記事タイトル
- SEOタイトル
- メタディスクリプション
- メインクエリ
- 記事情報補完済み
- 補完日時・補完エラー
- 記事ランク
- 備考

記事DBにない正規化URLだけを新規記事として追加します。

## Home / Daily Management State Update

Homeを記事DBのランク・作業状態に統一し、日次更新の初回起動確認、新規記事補完案内、未取得URLの安全な内部管理を追加しました。日次更新で記事タイトル・SEOタイトル・メタディスクリプション・メインクエリは上書きしません。
