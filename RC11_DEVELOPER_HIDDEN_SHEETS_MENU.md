# RC11 開発者用・内部シート表示メニュー

## 追加メニュー

`開発者用`

- 内部シートを表示
- 内部シートを非表示
- 内部シート一覧

## Product版で非表示にする方法

`Code.gs` の次の設定を変更します。

```javascript
var SBM_ENABLE_DEVELOPER_MENU = false;
```

関数を削除しなくても、利用者の上部メニューには表示されません。

## 利用者向けシート

- Home
- 今日の改善
- 改善効果
- 記事管理
- 改善履歴
- セットアップ

これ以外を内部シートとして一括表示・非表示します。
