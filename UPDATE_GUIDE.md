# Product 5.2.1 更新手順

## 既存利用者

1. スプレッドシートの **拡張機能 → Apps Script** を開きます。
2. 既存の`コード.gs`をProduct 5.2.1の内容で全置換します。
3. 保存してスプレッドシートを再読み込みします。
4. **SIMS-Blog-Manager → シートの作成・修復**を1回実行します。
5. **SIMS-Blog-Manager → バージョン情報**で`5.2.1`を確認します。

既存の記事DB、改善結果、改善履歴は継続利用できます。新しいスプレッドシートへの移行は不要です。

## 確認項目

- Homeが新レイアウトで表示される
- 上部メニューが6項目で表示される
- 改善履歴を開くと書式とチェックボックスが反映される
- 今日の改善が初期2件、最大6件
- 改善の推移が7日・14日・21日・28日の4回
