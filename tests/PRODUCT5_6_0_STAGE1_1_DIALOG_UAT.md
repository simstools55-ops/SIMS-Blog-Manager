# Product 5.6.0 Stage 1.1 日次処理ダイアログ UAT

## 確認項目

1. 日次処理ダイアログに未実施／本日取得済みの状態が表示される。
2. 「実行する」を押すとボタンが非表示になり、スピナーが即時表示される。
3. `sbmRunDailyFetchStageFromDialog` が1回だけ呼び出される。
4. 取得成功後、スピナーが停止し、取得行数・有効URL数・除外件数・所要時間が表示される。
5. 完了画面は自動で閉じず、「閉じる」で終了する。
6. 取得失敗時はエラー内容と「再実行する」「閉じる」を表示する。
7. ダイアログ下部に重複した「閉じる」ボタンが表示されない。
8. 生成HTML内のクライアントJavaScriptが構文検査に合格する。
