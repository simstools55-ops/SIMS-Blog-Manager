# Product 5.0 RC11 Selection Workflow

## 主な変更

- 利用者向け「記事DB」を「記事管理」へ変更
- シート順を Home → 今日の改善 → 効果測定 → 記事管理 → 改善履歴 に統一
- 上部メニューを次の順に整理
  - SIMS-Blog-Manager
  - 今日の改善操作
  - 改善効果操作
  - 記事操作
  - 改善履歴操作
- 今日の改善・効果測定・記事管理・改善履歴の左端に一択チェックボックスを追加
- 詳細表示等はチェックした1件を対象に実行
- シート作成・修復後のナビゲーターボタンを押すとダイアログを閉じる
- 新規追加記事は次回更新まで記事管理の最上部・淡い赤背景で表示
- 今日の改善のメインクエリを折り返し表示
- 効果測定予定日が空欄の場合、改善日と推奨確認日数から補完

## 制約

Googleスプレッドシートのカスタムメニューは、選択状態に応じたグレーアウトに対応していません。対象未選択または複数選択時は、1件を選択する案内を表示します。
