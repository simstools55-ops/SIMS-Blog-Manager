# 今日の改善

「今日の改善」は、初心者が候補の多さで迷わないよう、最初は2記事だけ表示します。

- 🟢 即効性：短時間の修正で効果が出やすい記事 1件
- 🔵 CTR改善：表示回数が多く、クリック率改善の効果が大きい記事 1件

シートには、改善するかどうかを判断するための最小限の情報だけを表示します。

- 区分
- 記事タイトル
- 改善理由と期待効果
- 予想時間
- 記事詳細

記事タイトルをクリックすると公開記事が開きます。「記事詳細」をクリックすると、改善ブリーフがポップアップ表示されます。

## 次のおすすめ

シート下部の「次のおすすめを表示（2記事追加）」をクリックすると、即効性とCTR改善から1件ずつ追加します。

- 初期表示：2件
- 1回目：4件
- 2回目：6件

6件すべての改善ブリーフは、候補作成時に準備済みです。追加表示時に再分析は行いません。

## 改善理由と期待効果

改善理由は2行で表示します。

1. 選ばれた理由
2. 期待できる効果

例：

> 順位7.2位で、短時間の修正でも伸びやすい位置です。  
> 期待効果：約+18クリックが見込めます。

期待クリック数はSearch Consoleの表示回数、現在CTR、掲載順位から算出した目安です。実際の結果を保証するものではありません。
