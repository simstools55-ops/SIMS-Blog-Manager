# 改善履歴

改善履歴は、実施した改善と4週間の判定結果を一覧で確認する画面です。

## 一覧に表示する項目

- 改善日
- 記事タイトル
- 改善概要
- 使用AI
- 1週
- 2週
- 3週
- 4週
- 最終判定
- 状態

一覧には測定日時を表示しません。各週の列には「改善」「大きく改善」「横ばい」「悪化」「未測定」のいずれかを表示します。

## 詳細画面

対象記事を1件選択し、「改善履歴」で改善履歴を開き、対象記事を選んで詳細を実行すると、次を確認できます。

- 改善計画と実施内容
- 改善前と現在の比較
- 1週目から4週目までの測定日時
- 各週の判定
- 各週のSIMS寸評
- 4週目終了後の最終総括
- 次の改善提案

1週目から4週目までのSIMS寸評は、数値の変化と推移だけを説明します。測定途中では新しい改善を提案しません。改善提案は4週目終了後の最終判定でのみ表示します。
