# 改善ブリーフ

改善ブリーフは、記事改善の作業指示書です。

## シート上に表示する項目

- 優先度
- 記事タイトル
- メインキーワード
- 改善内容
- 予想時間
- 期待効果
- 改善依頼（詳細）

シート上では、今日何を直すかだけを確認します。

## 改善依頼（詳細）

`改善依頼（詳細）` をクリックすると、次の内容を確認できます。

- Claude / ChatGPT に貼り付ける改善依頼文
- なぜ改善するのか
- 具体的な修正方法
- 修正例
- 参考クエリ
- FAQ候補
- クリック数、表示回数、CTR、平均順位

AIを使わず自分で修正する場合も、理由・具体例を見ながら作業できます。
