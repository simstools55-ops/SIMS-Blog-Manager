# 改善効果

改善した記事は、改善日から7日後・14日後・21日後・28日後の4回確認します。

## 基本操作

1. 上部メニューの「改善効果操作」から「開く」を選びます。
2. 最新データを反映するときは「更新」を選びます。
3. 詳しく確認したい記事を1件選択し、「詳細」を選びます。

旧版にあった「7日間延長」と「測定完了」の操作はありません。測定予定は自動で進み、4回目の記録後に自動で完了します。

## 改善効果シートの主な項目

- 次回測定予定日
- 測定回数（0回／4回～4回／4回）
- 改善前CTRと現在CTR
- 改善前順位と現在順位
- 最新判定
- 最新測定日時
- 測定状態

## 詳細画面

詳細画面は常に「改善日」と「現在」を比較します。前回測定との差ではなく、改善前から現在までの累積変化を確認できます。

## 測定の進み方

- 1回目：改善から7日後
- 2回目：改善から14日後
- 3回目：改善から21日後
- 4回目：改善から28日後

測定日を迎えた記事は、「改善効果操作 → 更新」を実行した時点の判定が改善履歴へ記録されます。
