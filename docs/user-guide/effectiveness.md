# 改善の推移

改善した記事を、改善日から7日・14日・21日・28日の4回確認します。

## 基本操作

1. Search Consoleデータと記事管理を最新にします。
2. 上部メニューの「SIMS-Blog-Manager → 改善記事の状況を確認」を選びます。
3. 画面を開く際に、測定対象の最新状態が反映されます。
4. 測定日を迎えた記事の結果が改善履歴へ記録されます。
5. 対象を1件選び「詳細」を実行すると、改善前と現在の比較と4週間の測定履歴を確認できます。

旧版にあった「7日間延長」と手動の「測定完了」はありません。測定予定は自動で進み、4回目の記録後に自動で完了します。

## 改善の推移シートの主な項目

- 次回測定予定日
- 測定回数
- 記事タイトル
- 経過日数
- 改善前CTR・現在CTR
- 改善前順位・現在順位
- 最新判定
- 測定状態

## SIMS寸評と最終判定

1週目から4週目までの寸評は、CTR・順位・クリック数・表示回数の変化を説明する観察コメントです。測定途中では改善提案を行いません。

4週目の測定完了後に、最終判定、4週間の総括、次の改善提案を表示します。
