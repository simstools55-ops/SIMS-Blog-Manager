# 効果測定

Product 5.0 Officialでは、改善した記事を週1回、最大2か月測定します。

## 測定期間

改善実施日から最大2か月です。

## 測定頻度

週1回です。

## 再修正した場合

改善中の記事を再修正した場合は、改善実施日を更新します。
そこから再度2か月測定します。

## 測定終了

2か月経過したら、改善ログへ保存して測定を終了します。
その後は通常運用に戻ります。
次回Search Consoleデータ取得時に、必要であれば再び改善候補へ入ります。

## 見るポイント

- 改善前順位と現在順位
- 改善前CTRと現在CTR
- 改善前クリックと現在クリック
- SIMS評価
- 次のアクション
