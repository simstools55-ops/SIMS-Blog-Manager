# 効果測定

RCテスト中は、14日・30日を待たずに、改善後7日間を毎日確認できます。

## 見る場所

- 効果測定：最新の結果を一覧で確認
- 測定履歴：改善後1日目から7日目までの記録を確認

## 判定の見方

- 改善傾向：順位、CTR、クリックのいずれかが良くなっています。
- 様子見：大きな変化はまだありません。
- 要確認：悪化傾向があるため、再確認します。
- 成功：7日経過後に改善が確認できた状態です。
- 要再改善：7日経過後に改善が見られない状態です。

## テスト終了後

正式運用では14日・30日の測定へ戻す予定です。


## RCテスト用：7日間の日次測定

Product 5.0 RC3では、正式運用前の確認用として、改善後7日間だけ毎日測定できます。

メニューから **今日の改善 → テスト用：今日の測定を記録** を実行すると、同じ日の重複を避けながら「測定履歴」へ現在値を記録します。

正式運用では、短期変動に左右されにくい **14日・30日測定** を基本にします。
