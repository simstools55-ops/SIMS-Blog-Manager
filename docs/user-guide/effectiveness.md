# 改善効果

改善効果では、改善前と現在のSearch Console指標を比較します。

## 4週間の測定スケジュール

改善後は、次の4回を自動的に測定します。

| 回数 | 測定時期 |
|---|---|
| 1回目 | 改善から7日後 |
| 2回目 | 改善から14日後 |
| 3回目 | 改善から21日後 |
| 4回目 | 改善から28日後 |

メニューの「改善効果操作 → 更新」を実行すると、測定日を迎えた記事の結果が改善履歴へ記録されます。
4回目の測定後は「測定完了」になります。

## 改善効果シート

改善効果シートには、次の情報を表示します。

- 次回測定予定日
- 測定回数（0回／4回～4回／4回）
- 改善前と現在のCTR
- 改善前と現在の掲載順位
- 最新判定
- 測定状態

## 詳細画面

詳細画面では、毎回「改善日」と「現在」を比較します。
前回測定時との差ではなく、改善前から現在までの累積変化を確認できます。

## 判定

- 大きく改善
- 改善
- 横ばい
- 悪化
- 測定待ち

7日延長機能はありません。7日・14日・21日・28日の4回を自動管理します。
