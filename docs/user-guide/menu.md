# メニューの使い方

Product 5.1 Officialでは、シート名ではなく「何をするか」で操作を選びます。

## SIMS-Blog-Manager

- **Homeを確認する**：ブログ全体と今日の状況を確認します。
- **今日の改善を開始**：今日の改善候補を開きます。
- **改善結果を登録**：AIの回答末尾にあるSIMS形式JSONを登録します。
- **改善記事の状況を確認**：改善後7日・14日・21日・28日の状況を確認します。
- **全ブログ記事を確認する**：記事一覧を開き、記事ランク・改善状態・クリック数・表示回数・CTR・掲載順位・最終取得日時で並び替えます。
- **改善推移を確認**：改善履歴と4回の測定結果を確認します。

## 管理

- **日次処理を実行**：起動時にスキップした場合や、もう一度最新データを取得したい場合に実行します。
- **初回セットアップ**：ブログ登録、API設定、Search Console接続、初期取得を進めます。
- **ブログ情報を変更**：登録ブログの情報を修正します。
- **記事情報を取得**：新規記事などのタイトル・SEOタイトル・メタディスクリプション・メインクエリを補完します。
- **シートの作成・修復**：必要シート、見出し、表示書式を修復します。
- **設定を開く**：利用者設定を確認・変更します。
