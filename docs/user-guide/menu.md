# メニューの使い方

Product 5.1.2 Officialでは、日常作業をGoogleスプレッドシート上部の独立メニューから選びます。

## 日常作業メニュー

- **今日の改善を開始**：今日の改善を開く、改善詳細を見る、表示件数を調整する
- **改善結果を登録**：選択した記事の改善結果を登録する
- **改善記事の状況を確認**：改善後の記事の状況を確認する
- **全ブログ記事を確認する**：記事一覧、記事詳細、改善案、並び替えを利用する
- **改善推移を確認**：改善の推移、改善履歴、選択記事の詳細を確認する

## SIMS-Blog-Managerメニュー

初期設定・データ更新・保守機能をまとめています。

- Homeを確認する
- 日次処理を実行
- 初回セットアップ
- ブログ情報を変更
- 記事情報を取得
- シートの作成・修復
- 設定を開く
- セットアップ画面を開く
- バージョン情報

配布版には「開発者用」メニューは表示されません。
