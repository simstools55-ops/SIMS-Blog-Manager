# 上部メニュー

上部メニューは毎日の作業順に並びます。

## SIMS-Blog-Manager

- Homeを確認する
- 日次処理を実行
- 初回セットアップ
- ブログ情報を変更
- 記事情報を取得
- シートの作成・修復
- 設定を開く
- セットアップ画面を開く
- バージョン情報

## 記事改善スタート

- 今日の改善を開く
- 選択記事の改善詳細を見る
- 次の2件を表示
- 初期2件に戻す

## 結果登録

- 選択記事の改善結果を登録

## 推移確認

- 改善の推移を開く
- 最新データで更新
- 選択記事の詳細を見る

## 記事一覧

- 記事一覧を開く
- 選択記事の詳細を見る
- 選択記事の改善案を見る
- 記事ランク、改善状態、クリック、表示回数、CTR、掲載順位、最終取得日時で並び替え

## 改善履歴

- 改善履歴を開く
- 選択した履歴の詳細を見る
- 選択記事の全履歴を見る
