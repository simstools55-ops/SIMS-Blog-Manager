# メニューの使い方

Product 5.1.3 Officialでは、毎日の作業を上部メニューから選びます。

## 上部メニューの並び

左から次の順です。

1. **SIMS-Blog-Manager**：Home、日次処理、セットアップ、シート修復、設定
2. **記事改善スタート**：今日の改善を開き、改善する記事を選ぶ
3. **結果登録**：改善後の結果を登録する
4. **推移確認**：改善中の記事と7日・14日・21日・28日の推移を確認する
5. **記事一覧**：全記事の確認、詳細表示、改善案、並び替え
6. **改善履歴**：終了済みを含む改善履歴の確認と詳細表示

## SIMS-Blog-Manager

- Homeを確認する
- 日次処理を実行
- 初回セットアップ
- ブログ情報を変更
- 記事情報を取得
- シートの作成・修復
- 設定を開く
- セットアップ画面を開く
- バージョン情報

## 記事改善スタート

- 今日の改善を開く
- 選択記事の改善詳細を見る
- 次の2件を表示
- 初期2件に戻す

## 結果登録

- 選択記事の改善結果を登録

## 推移確認

- 改善の推移を開く
- 最新データで更新
- 選択記事の詳細を見る

## 記事一覧

- 記事一覧を開く
- 選択記事の詳細を見る
- 選択記事の改善案を見る
- 記事ランク、改善状態、クリック数、表示回数、CTR、掲載順位、最終取得日時で並び替える

## 改善履歴

- 改善履歴を開く
- 選択した履歴の詳細を見る
- 選択記事の全履歴を見る

配布版では、開発者用メニューは表示されません。
