# 今日の改善

「今日の改善」は、今日実施する改善作業だけを見るための画面です。

## 表示件数

Product 5.0 Officialでは、改善候補30件の中から上位5件を表示します。

## 除外される記事

以下の記事は今日の改善に表示しません。

- 改善中
- 測定中
- 良好
- 改善不要

## 改善完了後の流れ

改善が終わったら、対象行を選択して「選択行を完了にする」を実行します。

完了すると次の処理が行われます。

1. 改善ログへ記録される
2. 改善中へ移動する
3. 今日の改善から除外される
4. 不足分が改善候補から補充される

## 使い方

1. 今日の改善を開く
2. 上から順に記事を開く
3. 改善内容を実施する
4. 実施した修正内容にチェックを入れる
5. 選択行を完了にする
