# 今日の改善

「今日の改善」は、毎日実行する改善タスクの一覧です。

通常はおすすめ上位5件だけを表示します。時間がある場合は、メニューから「改善候補をすべて表示する」を選ぶと全候補を確認できます。

## 画面の見方

|列|意味|
|---|---|
|優先|改善おすすめ度です。星が多いほど優先度が高いです。|
|時間|推定作業時間です。|
|記事タイトル|改善対象の記事です。|
|メインクエリ|改善の軸になる検索クエリです。|
|改善内容|最初に確認する改善ポイントです。|
|記事を開く|対象記事を開くリンクです。|
|詳細|チェックすると改善ブリーフを表示します。表示されない場合は行を選択してメニューから「選択行の改善ブリーフを開く」を使ってください。|
|完了|改善が終わったらチェックします。改善ログに自動記録されます。|

## 修正内容のチェック

改善が終わったら、完了チェックの前に、実施した内容へチェックを入れてください。

- Title
- H1
- Description
- 冒頭文
- H2/H3
- FAQ
- 内部リンク
- 本文追記
- その他

複数選択できます。これにより、あとから「どの改善が効果につながったか」を見やすくなります。

## 改善ブリーフ

改善ブリーフには、記事URL、メインクエリ、重要クエリ、本文補強クエリ、FAQ候補、改善理由、Claude/ChatGPTへ貼り付ける改善依頼文が含まれます。
