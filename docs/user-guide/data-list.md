# データ一覧

`データ一覧` は、Search Console取得データを改善管理用に整理して表示する一覧画面です。

表示項目は次の通りです。

| 項目 | 内容 |
|---|---|
| 記事ステータス | 良好 / 改善中 / 改善候補 / 様子見 / 管理対象外 |
| 記事URL | Search Consoleで取得した記事URL |
| H1タイトル | 記事本文側のH1タイトル |
| titleタグ | HTMLの `<title>...</title>` に入っているタイトル |
| メインクエリ | クリック数・表示回数から代表クエリを選定 |
| クリック数 | Search Console取得値 |
| 表示回数 | Search Console取得値 |
| CTR | クリック数 ÷ 表示回数 |
| 平均順位 | 表示回数で重み付けした平均順位 |

## 記事ステータス

- **良好**：改善の必要性が低い記事
- **改善中**：改善実施後、効果測定中の記事
- **改善候補**：これから改善する候補記事
- **様子見**：データが少なく、改善必要性を判断しにくい記事
- **管理対象外**：`#toc1` など、記事本体ではないURL

一覧は、`良好 → 改善中 → 改善候補 → 様子見 → 管理対象外` の順に並びます。

## 更新タイミング

`データ一覧` は、改善候補の分析・抽出が終わったあとに更新されます。

Search Consoleデータを取得しただけの段階では更新しません。
