初回認証について

初めてApps Scriptを実行するとGoogleの承認画面が表示されます。これは正常です。承認後に止まった場合は、同じSTEPをもう一度実行してください。

セットアップ手順

STEP1 ブログ情報を登録

  1. メニュー SIMS-Blog-Manager を開く
  2. セットアップ → STEP1 ブログ情報を登録 を実行
  3. ブログ名、ブログURL、Search Consoleプロパティを入力

STEP2 Google Cloud APIを有効化

  1. Apps Scriptの「プロジェクトの設定」でGoogle Cloudプロジェクト番号を確認
  2. Google Cloud Consoleで同じ番号のプロジェクトを開く
  3. Google Search Console APIを有効化
  4. 数分待ってスプレッドシートへ戻る

STEP3 Search Console接続テスト

成功すると日次データ取得が有効になります。

STEP4 初回データ取得

Search Consoleデータを取得し、今日の改善を作成します。
