# Home画面

Homeは、ブログの成長と今日の作業を一画面で確認するためのコンパクトなダッシュボードです。

## 基本情報

- ブログ名
- ブログURL
- 総記事数
- 最終更新日時

## 記事ランクのまとめ

記事ランクごとの件数と、前回の日次更新からの変化を表示します。

- 🏆 エース
- ✅ 安定
- 📈 成長
- 🌱 育成
- ⚠️ 低迷

変化は次の矢印で表示します。

- ↗：前回より増加
- →：前回と同じ
- ↘：前回より減少

## 改善状況

- 今日の改善
- 改善中
- 効果測定中
- 要確認

「要確認」は、Search Consoleで3回連続、または14日以上データを取得できなかった記事です。自動削除はされません。

## 今日のメッセージ

記事ランクと改善状況をもとに、ブログの現在地と今日の進め方を短い文章で表示します。

## 処理状況

Home下部は次の4項目だけを表示します。

- 処理名
- 開始時刻
- 処理結果
- お願い
