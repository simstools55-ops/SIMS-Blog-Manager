# FAQ

## 初回認証が出ました。大丈夫ですか？

正常です。スプレッドシート更新とSearch Consoleデータ取得のために必要です。承認後、同じSTEPをもう一度実行してください。

## Search Console APIを有効化したのに403になります

Apps Scriptが使っているGoogle Cloudプロジェクト番号と、APIを有効化したプロジェクト番号が違う可能性があります。エラー画面のプロジェクト番号と、Google CloudのIAMと管理に表示される番号を確認してください。

## Project Numberとは何ですか？

Google Cloudプロジェクトを識別する数字です。Apps ScriptがどのGoogle Cloudプロジェクトを使うかを確認するために必要です。

## API有効化後、どれくらい待てばいいですか？

通常は数分です。30分以上待っても変わらない場合は、プロジェクト番号違い・OAuth不足・Search Console権限不足を確認してください。

## 今日の改善が空です

管理対象割合・最低表示回数・最低クリック数の条件が厳しい可能性があります。また、改善直後の記事は14日間再提案されないことがあります。

## 改善ブリーフは何に使いますか？

ClaudeやChatGPTへ貼り付けることで、記事改善の材料として使えます。AIを使わない場合も、メインクエリ・参考クエリ・FAQ候補の確認に使えます。

## 全候補を見たいです

通常は5件表示です。メニューから全件表示に切り替えることで、改善候補をすべて確認できます。
