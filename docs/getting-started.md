# はじめに

## 必要なもの

- Googleアカウント
- 対象ブログのSearch Console閲覧権限
- Googleスプレッドシート

## 最短手順

1. テンプレートxlsxをGoogleスプレッドシートへインポート
2. Apps Scriptを開く
3. `Code.js` をコード.gsへ貼り付ける
4. `appsscript.json` を設定する
5. メニュー「セットアップウィザード開始」を実行
6. 設定シートB4にブログ名、B5にSearch Console Propertyを入力
7. 「初期設定を完了」
8. 「Search Console接続テスト」
