# ダウンロード

Product 5.0 Official Lean Base では、RC版で増えた不要なファイルをそぎ落とし、利用者が使うものだけに絞っています。

## 配布物

### スプレッドシートテンプレート

`spreadsheet/SIMS-Blog-Manager-template-Product5.0-Official-Lean.xlsx`

Google Drive にアップロードし、Googleスプレッドシートとして開いて使用します。

### Apps Script

`apps-script/コード.gs`

利用者に渡す Apps Script は、この1ファイルにまとめます。
Google Apps Script の「コード.gs」に貼り付けて使用します。

## 基本手順

1. ZIPをダウンロードする
2. スプレッドシートテンプレートをGoogle Driveへアップロードする
3. Googleスプレッドシートとして開く
4. Apps Scriptを開く
5. `コード.gs` の内容を貼り付ける
6. 初回セットアップを実行する

## 注意

Product 5.0では、以前のRC版を踏襲しながら、正式版に不要なものを減らします。
そのため、過去テンプレートや分割ソースは通常の利用者向け配布物には含めません。
