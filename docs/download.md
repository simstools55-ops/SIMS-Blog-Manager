# 製品本体の場所

Product 5.1 Officialの製品本体は、このZIP内にあります。

## スプレッドシート

`spreadsheet/SIMS-Blog-Manager.xlsx`

Googleドライブへアップロードし、Googleスプレッドシートとして開きます。

## Apps Script

`apps-script/コード.gs`

Apps Scriptの `コード.gs` に貼り付けます。

## 利用者が見る画面

通常表示するタブは次の5つです。

1. Home
2. 今日の改善
3. 改善中
4. ブログ診断
5. 処理ログ

管理用シートは非表示です。
