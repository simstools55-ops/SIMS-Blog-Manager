# スプレッドシート画面ガイド

## Home

毎日最初に見る画面です。
表示内容は次の3点です。

- ブログ全体の状況
- 改善作業の状況
- 今日やること

## 今日の改善

改善候補30件の中から上位5件を表示します。
改善が終わったら「選択行を完了にする」を実行します。

## 改善中

改善実施日、経過日数、現在の判定、次のアクションを確認します。
週1回、最大2か月の効果測定対象です。

## 改善ログ

改善した記事の履歴を保存します。
2か月の測定が終わった記事は、通常運用へ戻ります。

## 処理ログ

処理ごとに以下を記録します。

- 開始時刻
- 終了時刻
- 利用者待ち時間
- 対象件数
- 処理件数
- 件数
