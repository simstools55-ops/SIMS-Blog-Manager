# スプレッドシート画面

## データ一覧

データ一覧は、記事ステータス・記事タイトル・メインクエリ・主要指標だけを表示します。

SEOタイトル（titleタグ）、メタディスクリプション、記事URLなどの詳細は、データ一覧の「詳細」列をクリック、またはメニューの「選択行の詳細を表示」から確認します。


## 改善履歴の詳細ポップアップ

詳細画面は、次の3部構成です。

1. 改善計画
2. 実施した改善
3. 改善効果

「3. 改善効果」では、さらに次の順で確認します。

- 3-1. 改善前と現在の比較
- 3-2. 4週間の効果測定
- 3-3. 最終判定

未測定の週も、改善日から7日・14日・21日・28日後の予定日とともに表示されます。測定済みになると、測定日時・判定・SIMS寸評へ切り替わります。1～4週目の寸評には改善提案を含めず、改善提案は4週目終了後の最終判定にだけ表示します。
