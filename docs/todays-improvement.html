今日の改善

ホームセットアップ今日の改善改善ログ・効果測定FAQ

「今日の改善」は毎日使うタスクリストです。最初におすすめ2件を表示し、「次のおすすめ」で2件ずつ、最大6件まで追加できます。

列意味
優先改善おすすめ度です。
時間推定作業時間です。
記事タイトル改善対象の記事です。
メインクエリ改善の軸になる検索クエリです。
記事を開く記事URLを開きます。
詳細チェックすると改善ブリーフを開きます。開かない場合はメニューから開いてください。
完了改善後にチェックすると改善ログへ記録されます。

修正内容チェック

完了前に、実施した内容へチェックしてください。複数選択できます。

  • Title
  • H1
  • Description
  • 冒頭文
  • H2/H3
  • FAQ
  • 内部リンク
  • 本文追記
  • その他

改善ブリーフ

記事URL、メインクエリ、重要クエリ、本文補強クエリ、FAQ候補、改善理由、Claude/ChatGPTに貼れる改善依頼文を表示します。
