Search Console API has not been used in project...

Search Console APIが有効化されていない、またはプロジェクト番号が違う可能性があります。

  1. Apps Scriptの「プロジェクトの設定」を開く
  2. Google Cloudプロジェクト番号を確認
  3. Google Cloud Consoleで同じ番号のプロジェクトを開く
  4. Google Search Console APIを有効化
  5. 数分待つ
  6. STEP3接続テストを再実行

重要：番号が違う場合、30分待っても接続できません。

Googleの承認画面が出る

正常な動作です。許可後に処理が止まった場合は、同じSTEPをもう一度実行してください。

詳細チェックを入れても改善ブリーフが出ない

対象行を選択して、メニューから「選択行の改善ブリーフを開く」を実行してください。

完了チェックを入れても改善ログに入らない

URL列が空欄になっていないか確認してください。今日のデータを再取得すると直る場合があります。
