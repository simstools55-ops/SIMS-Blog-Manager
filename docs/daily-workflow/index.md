# 毎日の使い方

SIMS-Blog-Managerは、毎日30分だけブログ改善を進めるための道具です。

## 毎日の流れ

1. ホームを開きます。
2. 今日の改善を確認します。
3. 「詳細を見る」で改善ブリーフを開きます。
4. 記事URLを開きます。
5. 改善ブリーフをClaudeやChatGPTへ貼ります。
6. 記事を修正します。
7. 完了チェックを入れます。
8. 改善ログに記録されます。
9. 7日・14日・21日・28日の4回、改善の推移を確認します。

## まずは5件で十分

初期表示は5件です。時間がある場合は「すべて表示」を使って候補を広げます。

## 改善後に記録すること

完了時には、何を直したかを選びます。

- titleタグ
- H1
- メタディスクリプション
- 冒頭文
- H2/H3
- FAQ
- 内部リンク
- 本文追記
