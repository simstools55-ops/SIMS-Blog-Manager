# 毎日の使い方

## 1. Homeを確認

スプレッドシートを開いたら、最初にHomeで記事ランク、改善状況、今週の取り組みを確認します。

## 2. 日次処理

その日の日次処理が未実行の場合、起動時に確認画面が表示されます。スキップした場合や、後から再実行する場合は、**SIMS-Blog-Manager → 日次処理を実行**を選びます。

日次処理ではSearch Consoleデータ、記事ランク、改善候補、Homeを更新します。記事タイトルや改善履歴は保持します。

## 3. 記事改善スタート

**記事改善スタート → 今日の改善を開く**を選びます。

- 初期表示：2件
- 「次の2件を表示」で追加
- 最大表示：6件
- 「初期2件に戻す」でリセット

チェックボックスで1件選択し、**選択記事の改善詳細を見る**を実行します。

## 4. 結果登録

記事の改善が終わったら、**結果登録 → 選択記事の改善結果を登録**を選び、`SIMS_FEEDBACK_V1以降`のJSONを貼り付けます。

登録後は今日の改善から外れ、改善の推移確認へ移ります。

## 5. 推移確認

**推移確認 → 改善の推移を開く**から、改善後7日・14日・21日・28日の変化を確認します。

## 6. 記事一覧・改善履歴

今日の改善以外の記事を探す場合は**記事一覧**、終了済みの改善記録を見る場合は**改善履歴**を使います。
