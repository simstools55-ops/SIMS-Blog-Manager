# 毎日の使い方

Product 5.0 Officialでは、毎日の作業はHomeから始めます。

## 1. Homeを見る

Homeには次の3つだけが表示されます。

- ブログ全体の状況
- 改善作業の状況
- 今日やること

ブログ全体の状況では、良好記事数、改善中記事数、改善候補数を確認します。

## 2. その日最初の起動時

セットアップが完了している場合、その日最初の起動時のみ自動で以下を実行します。

1. Search Consoleデータ取得
2. 改善候補抽出

処理中は次のような進捗が表示されます。

- Search Consoleデータ収集中
- 改善候補抽出中

## 3. 今日の改善を見る

「今日の改善」には、改善候補30件の中から上位5件が表示されます。
上から順に改善してください。

## 4. 改善が終わったら完了にする

記事を改善したら、対象行を選択してメニューから「選択行を完了にする」を実行します。
完了した記事は改善中へ移動し、今日の改善から除外されます。
不足分がある場合は、改善候補から補充されます。

## 5. 効果測定を確認する

改善中の記事は、週1回、最大2か月測定します。
2か月経過したら改善ログへ保存して測定を終了します。
