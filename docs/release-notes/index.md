# リリースノート

## Product 5.0 Release 1 Sprint 5

- 4週間効果測定モデルを正式スキーマとして強制適用
- 旧「7日間延長」「測定完了」メニューを完全除去
- 改善履歴の旧「測定予定日」列を4回測定列へ移行
- バージョン情報メニューを追加
- シート作成・修復と起動時の両方で新仕様を適用

## Product 5.0 RC5

効果測定・カニバリ診断を「一覧＋詳細ポップアップ」方式へ改善しました。

- 効果測定は、記事タイトル・改善日・改善内容・判定・SIMS評価・次のアクションを中心に表示します。
- 詳しい数値や判定理由は「詳細」にチェックを入れて確認します。
- カニバリ診断も一覧では要点だけを表示し、詳細ポップアップで理由と対応策を確認します。
- deploy.yml の更新は不要です。

## Product 5.0 Release 1 Sprint 4

- 効果測定を改善後7日・14日・21日・28日の4回方式へ変更しました。
- 「7日延長」を廃止しました。
- 改善履歴へ4回分の測定日時と判定を保存します。
- 4回目の測定後は自動的に測定完了になります。
