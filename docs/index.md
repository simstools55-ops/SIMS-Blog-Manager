SIMS-Blog-Managerは、Google Search Consoleのデータを使って、今日改善すべき記事と改善に必要な材料を整理するGoogleスプレッドシートです。

<div class="sbm-important">
初めて使う場合は、まず <strong>はじめに → セットアップ → Search Console接続テスト</strong> の順に進めてください。
</div>

## 毎日使う画面

| 画面 | 役割 |
|---|---|
| ホーム | 今日の状態と次にやることを確認します。 |
| 今日の改善 | 改善候補を確認します。通常は上位5件で十分です。 |
| 改善ブリーフ | ClaudeやChatGPTへ貼る改善材料を確認します。 |
| 改善ログ | 何を直したか記録します。 |
| 効果測定 | 改善後の順位・CTR・クリック変化を確認します。 |
