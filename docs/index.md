# SIMS-Blog-Manager

**正式バージョン：5.1.3**

Google Search Consoleのデータから、今日改善する記事を選び、改善結果と4週間の推移を管理するGoogleスプレッドシート製品です。

## 毎日の基本フロー

1. Homeでブログの状態を確認
2. **記事改善スタート**で今日の記事を選択
3. 記事を改善
4. **結果登録**で改善結果を登録
5. **推移確認**で7日・14日・21日・28日の結果を確認
6. 必要に応じて**記事一覧**と**改善履歴**を確認

## 主な画面

- **Home**：記事ランク、改善状況、今週の取り組み
- **今日の改善**：初期2件、最大6件から改善記事を選択
- **改善の推移**：改善中の記事と4回の測定状況
- **記事管理**：全ブログ記事の検索データと状態
- **改善履歴**：終了済みの改善記録と詳細
