# SIMS-Blog-Manager Product 5.1.3 Official

SIMS-Blog-Managerは、Google Search Consoleのデータから改善候補を選び、改善作業と4週間の効果をGoogleスプレッドシートで管理するツールです。

**正式バージョン：5.0.1**

## 初めて利用する方

1. [はじめに](getting-started/index.md)
2. [セットアップ全体の流れ](setup/index.md)
3. [毎日の使い方](daily-workflow/index.md)

## 主な画面

- **Home**：ブログ全体の状態、今日の作業、処理状況を確認
- **今日の改善**：優先して改善する記事を選択
- **記事管理**：記事情報とSearch Console指標を確認
- **改善履歴**：改善計画、実施内容、改善の推移を確認
- **改善の推移**：7日・14日・21日・28日の測定状況を確認

## 基本の流れ

1. 日次更新でSearch Consoleデータを取得します。
2. 「今日の改善」から対象記事を選びます。
3. 改善ナビの内容を使って記事を修正します。
4. 実施した改善内容を記録します。
5. 4週間の測定結果を改善履歴の詳細画面で確認します。

困ったときは、[FAQ](faq/index.md) または [トラブルシューティング](troubleshooting/index.md) を確認してください。
