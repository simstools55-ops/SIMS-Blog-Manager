# Google Cloud API設定

Search Console APIを使うため、Google Cloud側でAPIを有効化します。

## 手順

1. メニュー「SIMS-Blog-Manager」→「セットアップ」→「STEP2 Google Cloud API設定ガイド」を押す
2. 表示されたポップアップのリンクからGoogle Cloudを開く
3. Search Console APIを有効化する
4. スプレッドシートに戻る
5. STEP3 接続テストを実行する

## 注意

Google Cloudを開いた後、セットアップは自動で次へ進みません。これはApps Scriptの仕様です。

外部URLを開いた後は、必ずスプレッドシートへ戻って次のSTEPを実行してください。
