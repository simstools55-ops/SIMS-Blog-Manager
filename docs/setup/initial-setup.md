# 初期セットアップ

Product 3.0では、セットアップはチェックリスト形式です。

## 手順

1. スプレッドシートを開く
2. メニュー「SIMS-Blog-Manager」を開く
3. 「セットアップ」→「STEP1 ブログ情報を入力」を押す
4. ポップアップに従い、以下を入力する
   - ブログ名
   - ブログURL
   - Search Consoleプロパティ

## Search Consoleプロパティの例

ドメインプロパティ：

```text
sc-domain:example.com
```

URLプレフィックス：

```text
https://example.com/
```

Search Consoleに表示されている表記をそのまま入力してください。
