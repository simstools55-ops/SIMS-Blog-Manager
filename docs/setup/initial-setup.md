# 初期セットアップ

Product 3.2では、セットアップはチェックリスト形式です。

## 手順

1. スプレッドシートを開く
2. メニュー「SIMS-Blog-Manager」を開く
3. 「セットアップ」→「STEP1 ブログ情報を入力」を押す
4. ポップアップに従い、以下を入力する
   - ブログ名
   - ブログURL
   - Search Consoleプロパティ

## Search Consoleプロパティの例

ドメインプロパティ：

```text
sc-domain:example.com
```

URLプレフィックス：

```text
https://example.com/
```

Search Consoleに表示されている表記をそのまま入力してください。


## 初回認証が表示された場合

初めてApps Scriptを実行すると、Googleの認証画面が表示されます。これはスプレッドシートとSearch Consoleを利用するために必要です。

1. 表示内容を確認します。
2. 「詳細」を開きます。
3. 「安全ではないページに移動」と表示される場合は、作成者が自分自身であることを確認して進みます。
4. 「許可」を押します。
5. スプレッドシートに戻り、同じSTEPをもう一度実行します。
