# 初期セットアップ

## 初回認証について

初めてApps Scriptを実行すると、Googleの承認画面が表示されます。これは正常です。

承認後に処理が止まったように見える場合は、同じSTEPをもう一度実行してください。

## 手順

1. メニュー `SIMS-Blog-Manager` を開く
2. `セットアップ` → `STEP1 ブログ情報を登録`
3. ブログ名を入力
4. ブログURLを入力
5. Search Consoleプロパティを入力
6. `STEP2 Google Cloud APIガイドを開く` を実行
7. `STEP3 Search Console接続テスト` を実行
8. `STEP4 初回データ取得` を実行

## Search Consoleプロパティの例

URLプレフィックスの場合：

```text
https://example.com/
```

ドメインプロパティの場合：

```text
sc-domain:example.com
```
