# 初回認証

初めてメニューを実行すると、Googleの承認画面が出ます。

## これは正常です

SIMS-Blog-Managerが次の操作を行うために必要です。

- スプレッドシートを更新する
- Search Consoleのデータを読み取る
- 画面に案内を表示する

## 手順

1. メニューからSTEPを実行します。
2. Googleの承認画面が出たら、利用するGoogleアカウントを選びます。
3. 「詳細」や「安全ではないページへ移動」と表示された場合は、作成者が自分であることを確認して進みます。
4. 許可します。
5. スプレッドシートに戻り、同じSTEPをもう一度実行します。

## 注意

承認後に自動で続きが始まらない場合があります。その場合は、同じメニューを再実行してください。
