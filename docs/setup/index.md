# セットアップ全体の流れ

セットアップは、メニュー **SIMS-Blog-Manager** からSTEP順に進めます。

## STEP一覧

| STEP | 内容 |
|---|---|
| STEP1 | ブログ情報を入力 |
| STEP2 | Google Cloud APIガイドを確認 |
| STEP3 | Search Console接続テスト |
| STEP4 | 初回データ取得 |

## 重要

初めてApps Scriptを実行すると、Googleの承認画面が表示されます。これは正常な動作です。承認後、同じSTEPをもう一度実行してください。
