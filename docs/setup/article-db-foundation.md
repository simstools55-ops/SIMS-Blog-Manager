# 記事DBの初回セットアップ

記事DBはSIMS-Blog-Managerの基礎データです。初回だけ、URL収集と記事情報補完を分けて実行します。

## STEP4 記事DB初期構築

Search Consoleからページデータを100件ずつ取得します。

- URL
- クリック数
- 表示回数
- CTR
- 掲載順位
- 記事ステータス

URLを正規化し、ノイズを除外して記事DBへ追加します。取得件数が100件未満になった時点で、最後まで取得したと判定します。途中で終了しても、同じメニューを再実行すると続きから再開します。

## STEP5 記事情報補完

URL収集の完了後、未補完の記事を50件ずつ処理します。

- 記事タイトル（H1相当）
- SEOタイトル（titleタグ）
- メタディスクリプション
- メインクエリ

280秒を超える前に安全終了し、取得済み情報を保存します。未補完記事が残っている場合は、同じメニューを再実行してください。

## 進捗の確認

Homeの「初回セットアップ進捗」または、セットアップメニューの「セットアップ進捗を確認」で確認できます。
