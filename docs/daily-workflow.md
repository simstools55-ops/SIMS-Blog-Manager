# 日次運用

## 1日の最初の起動

その日に初めてスプレッドシートを開くと、日次更新を実行するか確認する画面が表示されます。

- **日次更新を実行**：Search Consoleの最新ページデータを取得します。
- **今日は実行しない**：その日は再表示しません。翌日に再度確認します。

## 日次更新で更新する情報

- クリック数
- 表示回数
- CTR
- 掲載順位
- 記事ランク
- Homeの集計

記事タイトル、SEOタイトル、メタディスクリプション、メインクエリ、ArticleID、作業状態は保持します。

## Search Consoleに出なかった記事

その日の取得結果にURLがなくても、記事DBから削除しません。システム内部で連続未取得日数を管理し、30日以上続いた件数だけをHomeへ表示します。利用者が確認するまでは自動削除しません。

## 新規記事が見つかった場合

日次更新後に新規記事が見つかると、記事情報を取得するか確認する画面が表示されます。

取得対象：

- 記事タイトル
- SEOタイトル
- メタディスクリプション
- メインクエリ

「あとで」を選んだ場合は、Homeの「記事情報未取得」に件数が残ります。
