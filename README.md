## 更新手順

1. `apps-script/Code.gs` を全置換して保存
2. スプレッドシートを再読み込み
3. `管理 → シートを作成・修復` を1回実行
4. `データ更新 → 記事ランクを再判定` を実行
5. 記事DBの「記事ランク」「作業状態」を確認

## 記事DB詳細の開き方

1. 「記事DB」シートで確認したい記事の行を選択します。
2. 上部メニューから `SIMS-Blog-Manager → 記事DB → 選択記事の詳細を開く` を実行します。
3. 「記事DB詳細」ポップアップに、検索数値・記事ランク・作業状態・着手判断・SEO情報が表示されます。

セル選択だけでダイアログを開く方式はGoogleスプレッドシート上で安定しないため、選択行＋メニュー方式に統一しています。
