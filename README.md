
## Operation Refactoring Stage 1

旧改善機能の起動経路と旧シートを停止し、記事DBだけを参照する新しい「今日の改善」「改善ブリーフ」を再構築するための整理版です。現段階では旧おすすめ改善は表示しません。
# SIMS-Blog-Manager Product 5.0.0 Article Rank / Work State

記事DBの「記事ステータス」を、記事の価値を表す **記事ランク** と、現在の作業状況を表す **作業状態** に分離した更新版です。

## 記事ランク

- 🏆 エース
- 📈 成長
- ✅ 安定
- 🌱 育成
- ⚠️ 低迷

Search Consoleのクリック数・表示回数・CTR・掲載順位だけを使って再判定します。外部ページ取得は行いません。

## 作業状態

- 未着手
- 🔥 今日の改善
- ✏️ 改善中
- 👀 モニター中
- ✔️ 完了

日次更新では作業状態を上書きしません。

## 更新手順

1. `apps-script/Code.gs` を全置換して保存
2. スプレッドシートを再読み込み
3. `管理 → シートを作成・修復` を1回実行
4. `データ更新 → 記事ランクを再判定` を実行
5. 記事DBの「記事ランク」「作業状態」を確認
