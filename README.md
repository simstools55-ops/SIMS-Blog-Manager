# SIMS-Blog-Manager Product 5.0.0

## Settings Numeric Controls

設定シートを数値入力方式へ整理した更新版です。

### 主な変更

- 記事情報補完件数を30～100の範囲で自由入力
- 今日の改善の初期表示件数・最大表示件数を設定可能
- 改善候補抽出件数を設定可能
- Search Console取得期間を設定可能
- 範囲外・小数・文字列入力を防止

既存スプレッドシートをそのまま継続利用できます。


## 記事ランク試算（開発確認）

記事DBに保存済みのSearch Console指標だけを使い、記事の検索貢献度を5段階で試算します。外部ページ取得は行いません。

- 🏆 エース
- 📈 成長
- ✅ 安定
- 🌱 育成
- ⚠️ 低迷

実行場所：`SIMS-Blog-Manager → データ更新 → 記事ランクを試算（開発確認）`

この判定は検索データ上の貢献度であり、実収益そのものを示すものではありません。
