# SIMS-Blog-Manager Product 5.0 Official

Product 5.0 Officialは、RC9をベースに不要なものをそぎ落とし、必要なものだけに絞った正式版です。

## 製品本体

- スプレッドシート：`spreadsheet/SIMS-Blog-Manager.xlsx`
- Apps Script：`apps-script/コード.gs`

## 利用者が見るタブ

- Home
- 今日の改善
- 改善中
- ブログ診断
- 処理ログ

管理用シートは非表示です。

## 使い始める手順

1. `spreadsheet/SIMS-Blog-Manager.xlsx` をGoogleドライブへアップロード
2. Googleスプレッドシートとして開く
3. Apps Scriptを開く
4. `apps-script/コード.gs` の内容を貼り付ける
5. 初回セットアップを実行する
